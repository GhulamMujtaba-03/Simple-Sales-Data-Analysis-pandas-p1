# Sales Data Analysis Using Python & Pandas

This project demonstrates **data exploration, analysis, and insights** on a sales dataset using Python and Pandas.

## Columns:
OrderID, Customer, City, Category, Product, Quantity, Price, Sales, Date

## Analysis Includes:
- Data Understanding (EDA)
- Basic Sales Analysis
- City Analysis
- Category Analysis
- Product Analysis
- Customer Analysis
- Time Analysis

## How to Run:
Open the notebook `Sales_Analysis.ipynb` in Jupyter and run all cells.
